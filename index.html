How to Use the Photo Crop & Signature Resizer Tool

Easily resize images for documents and exam applications by following these three simple steps:

1

Set Image Specifications

Define the required dimensions, including height, width, and file size, as per the application guidelines. For reference, check the table below for image requirements of major exams. For instance, if you need an image for the IBPS PO Exam, enter 200 pixels (or cm) for height and 300 pixels (or cm) for width, then proceed to download.

2

Upload Your Image

Select and upload the image or document that needs resizing. Use the provided adjustment tools to align the image within the designated frame.

3

Preview & Verify

Before downloading, preview the image to ensure it is correctly cropped and meets the required dimensions and file size.

4

Download the Resized Image

Once verified, download the optimized image, ready for submission with your exam or document application.
